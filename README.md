# SakiSaki

一个可以使角色在地面上蠕动并发出奇怪的叫声(saki酱~)的mod.

致敬: https://www.bilibili.com/bangumi/play/ep1231523

### Feature

在按下 [X] 键后，角色将会呈爬行姿态，再次按下以取消。

![爬行状态截图](https://cdn.modrinth.com/data/cached_images/b99a8512e2035065298b5e753232f7b63abeb317.png)

你也可以在联机的时候使用，旁边的人会听到你爬行的声音。

![联机爬行截图](https://cdn.modrinth.com/data/cached_images/31914d78e82fa07a5c6712c78f4cf34e6472c3f8.png)

你可以修改Saki酱~的声音

![声音设置界面](https://cdn.modrinth.com/data/cached_images/9b6ef3f9c6b036a6acd62e1002080774b1f6814a.png)

这个模组也兼容ysm

![ysm爬行动作](https://cdn.modrinth.com/data/cached_images/9e01f3d048c34bbbce97fbe2dcf125d1e4dd9286.png)
